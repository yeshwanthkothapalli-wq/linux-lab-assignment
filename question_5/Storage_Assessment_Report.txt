5. Storage Health Observations:
Both disk space and inode utilization are at healthy 1% usage levels, indicating no immediate storage concerns. The system has 955GB of available space and over 67 million free inodes.

6. Recommendations for Improving Storage Utilization:
- Set up regular monitoring alerts (e.g., using df cron jobs) to catch usage before it reaches 80-90%.
- Since usage is currently very low, no immediate action is needed, but periodic checks should be scheduled as the application server begins storing data.
- Consider separating application data and logs onto dedicated partitions to prevent one growing directory from affecting overall root filesystem health.
